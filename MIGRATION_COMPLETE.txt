================================================================================
TROUBLESHOOTING
================================================================================

SSL Certificate Not Found:
   → Re-download with curl command above
   → Verify: ls -la $HOME/.postgresql/root.crt

Connection Refused:
   → Run: npm run db:test
   → Check: echo $DATABASE_URL
   → Verify: SSL mode is verify-full

No Tables After Migration:
   → Run: npm run db:migrate
   → Check: psql "$DATABASE_URL" -c "SELECT count(*) FROM information_schema.tables"

Import Errors:
   → Find remaining Supabase imports
   → Replace with CockroachDB equivalents
   → Ensure correct file path

See QUICKSTART_MIGRATION.md for more troubleshooting.
